## 8. 분석

### 서비스 구조 이해
- **웹**: 주 가입/이용 채널 — 신규 유저는 웹에서 온보딩 → 가입 → 예측/응모
- **앱**: 웹 유저의 앱 전환 유도 채널 — 웹 계정으로 앱 로그인 가능, 앱 신규 가입 불필요
- **4/6 앱 폭증 구조**: KBO 티켓 당첨 알림톡(kakao_notitalk) 랜딩을 앱으로 설정 → 기존 웹 유저들이 당첨 결과 확인하러 앱 첫 진입 → 앱 설치 유도 전략 작동

### 인사이트
- **알림톡 → 앱 설치 유도 전략 효과 입증**: 당첨 결과 알림톡 랜딩을 앱으로 설정한 전략이 작동. 기존 웹 유저 830명 앱 진입 + 온보딩 667명 노출. 다음 당첨 발표 시 동일 전략 반복 집행 — 앱 MAU 누적 가속화 가능
- **앱 전환 유저 서비스 이용률 높음**: 앱 진입 후 예측 66.9%, 응모 32.0% — 당첨 결과 확인 목적으로 들어왔지만 서비스 참여까지 이어진 비율 높음. 앱 응모 미완료 유저(526명) 대상 D+1 앱 Push로 추가 전환 가능
- **웹 온보딩 최대 병목 확인**: 인트로→시작하기 이탈 **64.9%** — 웹 유입 유저의 3분의 2가 인트로에서 이탈. 인트로 카피/비주얼 개선이 인트로→가입 CVR(현 15.5%) 개선의 가장 큰 레버
- **웹 신규 가입 채널 공백**: 웹 신규 가입 115명(▼-7%) — round_push·instagram 소진. 앱 전환 활성화와 별개로 웹 신규 유입은 지속 감소 중. round 2차 푸시 미집행 시 다음 주 신규 가입 급감 예상
- **kakao_opentalk 고효율**: CVR 34.9% — 오픈채팅 인플루언서 추가 발굴 가속. 신규 유입 채널 공백 메울 수 있는 가장 즉각적인 대안
- **paid 소재 교체 판단**: 4/6 CVR 5.4%는 시야 명당(~16시) + 프리미엄 석(16시~) 혼합 성과. 프리미엄 석 단독 효과는 4/7에 확인 — 8% 기준 미달 시 확정 예매권 롤백 검토
- **웹 응모/signin 9.7%**: 전일 15.3% 대비 ▼-5.6pp — 당일 신규 가입자가 적고 앱 전환 위주 날의 구조적 특성. 이상 수치 아님
