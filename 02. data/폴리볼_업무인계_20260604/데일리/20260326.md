## 8. 분석

### 팩트
- DAU 1,267명 (웹 1,259 / 앱 8) — 론칭 첫날 기준값 설정
- 가입: 에어브릿지 428명 / 서버 489명 (+14.3% 차이)
- 온보딩 최대 이탈 구간: 로그인 바텀시트→가입완료 62.7% (681→427) — 소셜 로그인 부재가 주원인
- 예측/signin 62.9% (360/572) — 가입자 10명 중 6명이 당일 예측, 1인당 평균 2.78회
- 응모: 에어브릿지 19명(3.3%/signin) vs 서버 363명 — 19배 차이 ⚠ 이벤트 정의 불일치 의심
- 채널: instagram 831클릭·가입 0, round 437클릭·가입 0 → UTM 미연결 → unattributed에 428가입 귀속

### 인사이트
- 론칭 첫날 트래픽 퀄리티 최고 (인트로→팀선택 68.1%) — 이후 하락 추세 기준점
- 에어브릿지 응모완료(19명)와 서버 응모User(363명)의 19배 차이는 이벤트 트리거 시점 차이 또는 구현 오류 가능성 — 개발팀 응모 이벤트 정의 확인 필요
- instagram·round가 론칭 첫날 유입의 80% 이상 담당했으나 UTM 미연결로 성과 측정 불가 — 채널별 CPA 기준값 확보 실패
