## 8. 분석


### 팩트
- WAU 3,272명 (웹 3,230 / 앱 42) — 론칭 4일(3/26~3/29) 기준, 일별 DAU 합산: 3,573명 (웹 3,522 / 앱 51)
- 주간 누적 가입: 에어브릿지 811명 / 서버 1,175명 (+44.9% 차이) — 이벤트 귀속 갭 확인 필요
- 온보딩 합산: 인트로 3,177명 → 팀선택 57.0% → BS01 84.9% → 가입완료 52.8%
- 주간 예측완료 633명(에어브릿지) / 예측User 1,119명(서버) — 서버가 1.8배 높음
- 주간 응모완료: 에어브릿지 74명 vs 서버 응모User 1,049명 → 14.2배 차이 ⚠
- 서버 퀴즈User 1,007명 — 예측User 1,119명과 유사, 퀴즈 1인당 10.8회(Cnt 10,863/User 1,007) vs 예측 3.4회(3,835/1,119)
- 채널: round 2,297클릭/167가입(CVR 7.3%) 압도적 1위, instagram 1,161클릭/34가입(CVR 2.9%)
- unattributed 612가입 = 전체의 75.5% — UTM 미귀속 대부분은 D+1 UTM 연결 전 가입

### 인사이트
- **온보딩 CVR 하락 추세**: 인트로→팀선택 68.1%(3/26) → 47.8%(3/28) — 론칭 효과 소진 후 트래픽 퀄리티 저하. BS01→가입 52.8%는 소셜 로그인 부재가 상수 이탈 요인
- **응모 전환율 우상향**: 응모/signin 3.3%(3/26) → 9.8%(3/29) — 가입 D+1~2 후 재방문해 응모하는 지연 전환 패턴 확인. 리텐션 초기 흐름은 긍정적
- **퀴즈 반복 참여율 3.2배**: 퀴즈 1인당 10.8회 vs 예측 3.4회 — 퀴즈가 예측보다 반복성 강함. 퀴즈→예측→응모 전환 흐름이 리텐션 핵심 루프일 가능성
- **서버 vs 에어브릿지 응모 갭(14.2배)**: 에어브릿지 pv_match_application_completed가 실제 응모의 7%만 포착 — 이벤트 트리거 시점 또는 구현 오류. 응모 지표는 현재 서버 데이터 기준으로 봐야 함
- **unattributed 75.5%**: 첫날 UTM 미연결로 인한 대규모 미귀속 — 채널별 CPA 측정 신뢰도 낮음. 이후 round CVR(7.3%)이 실질적 기준값
