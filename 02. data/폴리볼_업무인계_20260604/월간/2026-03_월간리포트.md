## 9. 월간 분석

### 팩트 (6일, 3/26~3/31 기준)
- MAU 4,333명 (웹 4,241 / 앱 92) — 일별 DAU 합산: 4,940명 (웹 4,815 / 앱 125)
- 가입: 에어브릿지 913명 / 서버 1,035명 (+13.4%)
- 온보딩 합산 (웹): 인트로 4,253명 → 팀선택 52.5% → BS01 83.4% → 가입완료 49.1% → 예측CTA 77.6%
- 예측완료: 에어브릿지 843명 / 서버 User 940명
- 퀴즈 참여(서버): User 840명 / 1인당 10.4회 (퀴즈Cnt 8,707 / User 840)
- 응모: 에어브릿지 173명 vs 서버 User 926명 → 5.4배 차이 ⚠
- 채널: round 3,207클릭/232가입(CVR 7.2%) 1위, instagram 1,328클릭/53가입(CVR 4.0%)
- unattributed 630가입 = 에어브릿지 전체의 69.0%
- kakao_notitalk 217클릭/1가입 — 재활성화 캠페인(3/31), 응모/signin 27.0% 최고점 기록

### 온보딩 퍼널 분석
- **인트로→팀선택**: 68.1%(D+1) → 37.4%(D+6) 지속 하락 — 론칭 초기 관심층 소진. 인트로 콘텐츠 또는 타겟 개선 없이는 반등 어려움
- **BS01→가입완료**: 52.8% 평균 — 소셜 로그인 부재가 고정 이탈 요인. 카카오 로그인 도입 시 10~15pp 개선 기대 가능
- **온보딩 전체 효율(인트로→예측CTA)**: 인트로 4,253명 중 예측CTA 도달 607명(14.3%) — 인트로와 가입 사이에서 85.7%가 이탈

### 채널 분석
- **round 독주**: 클릭 3,207 / 가입 232 / CVR 7.2% — 론칭 전체 기간 압도적 1위. 그러나 paid 채널 미집행 기간이라 비교 기준 부재
- **instagram**: 1,328클릭/53가입(CVR 4.0%) — 팬페이지(naora9 CVR 30.4%) > 오운드미디어(CVR 1.1%) 큰 차이
- **unattributed 69%**: D+1 UTM 미연결 + 직접 방문이 혼재. 실제 채널 기여는 측정 불가 구간 존재

### 데이터 이슈
- 에어브릿지 응모 vs 서버 응모: 5.4배 차이 지속 — 에어브릿지 응모 지표 신뢰 불가, 서버 기준 사용 권장
- 에어브릿지 가입 vs 서버 가입: 일관되게 +10~15% 차이 — 온보딩 가입 이벤트 이중 발화 또는 계정 연동 로직 차이 가능성
