# 폴리볼 대시보드 업데이트 인계 매뉴얼

> Claude Code로 작업. 매일 명령 1줄로 끝.

---

## 0. 받은 zip 안에 있는 것

```
폴리볼_업무인계.zip
├── CLAUDE.md                          ← Claude Code 작업 흐름
├── dashboard/
│   ├── fetch_daily.py                 ← Airbridge fetch
│   ├── daily_pipeline.py              ← Airbridge 파이프라인
│   ├── fetch_ad_revenue.py
│   ├── backup_airbridge.py
│   ├── add_daily_costs.py             ← costs 자동 입력
│   ├── app.py                         ← 회사 대시보드 코드
│   ├── requirements.txt               ← 패키지 목록
│   └── 인계_매뉴얼.md                  ← 이 문서
└── .streamlit/
    └── secrets.toml                    ← Airbridge 토큰
```

> 데이터 파일(data.json, 일별 리포트)은 zip에 없음. **매일 회사 대시보드에서 다운로드해서 쓰면 됨.**

## 1회 셋업 (처음 한 번, 약 15분)

### Python 설치
1. https://www.python.org/downloads/ → **Python 3.12** 다운로드
2. 설치 시 **"Add Python to PATH" 체크 필수**
3. cmd로 확인: `python --version`

### 작업 폴더 셋업
1. zip 받아서 압축 풀기
2. 적당한 위치에 배치 (예: `C:\polyball_handover\`)
3. cmd 열고:
   ```
   cd C:\polyball_handover
   pip install -r dashboard\requirements.txt
   ```

### Claude Code 설치
- https://claude.com/claude-code 에서 설치

---

## 매일 작업 흐름 (10분)

### Step 1. Claude Code 켜기

cmd:
```
cd C:\polyball_handover
claude
```

Claude Code 켜지면 메시지 입력:

```
폴리볼 [어제날짜] 대시보드 데이터 업데이트해줘.

[자리어때] 노출 [숫자] 비용 [숫자]
[텐핑] 어드민 가입 [숫자] (CPA 500원)
[애드팝콘] [엑셀파일경로]
```

**예시**:
```
폴리볼 4/30 대시보드 데이터 업데이트해줘.

[자리어때] 노출 50000 비용 166666
[텐핑] 어드민 가입 50 (CPA 500원)
[애드팝콘] C:\Users\jane\Downloads\20260424_20260430.xlsx
```

→ Claude Code가 `CLAUDE.md` 보고 알아서:
1. `fetch_daily.py` 실행 (Airbridge 데이터)
2. 자리어때/텐핑 비용 추가
3. 애드팝콘 매출 갱신
4. 일별 md 분석 작성 (팩트 + 인사이트)
5. `폴리볼_데이터/2026-04-30_대시보드/` 폴더에 업로드용 파일 정리

### Step 2. 회사 대시보드 업로드

1. 회사 대시보드 마스터 로그인
2. 위에서 생성된 `폴리볼_데이터/2026-04-30_대시보드/` 폴더 안의 파일 2개 업로드:
   - `data.json`
   - `20260430.md`
3. 새로고침해서 분석 정상 표시되는지 확인

## 주간 리포트 (월요일에)

매주 월요일 = 지난 주(월~일) 주간 리포트.

```
폴리볼 W18 주간 리포트 만들어줘 (4/27 ~ 5/3).
```

→ Claude Code가 자동 생성 + 폴리볼_데이터 폴더에 정리.
→ 회사 대시보드의 `weekly/` 위치에 업로드.

## 자주 발생하는 문제

| 증상 | 해결 |
|---|---|
| `ModuleNotFoundError` | `pip install -r dashboard\requirements.txt` 다시 |
| Airbridge 401/403 | `.streamlit/secrets.toml` 토큰 만료 → 인계자에게 새 토큰 |
| 서버 데이터 0 | 30분 후 재시도 / 안 되면 개발팀 |
| Claude Code 멈춤 | 같은 명령 다시 입력 |
| data.json 없음 | zip의 data.json 위치 확인 (`dashboard/data.json`) |
