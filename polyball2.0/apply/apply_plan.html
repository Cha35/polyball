0핵심 요약

경품 응모는 KBO 직관 티켓에 이은 두 번째 보상 트랙으로, ① 야구 시즌 종료 시 광고 매출 하락 리스크를 방어하고 ② 볼(P) 사용을 촉진해 광고 노출(BM)을 확대하기 위한 서비스다.

① 참여는 대체 가능
월드컵(비KBO 예측→볼→경품 응모)이 KBO의 75% 규모로 실제 작동. "야구 없으면 이탈" 우려는 데이터상 약함.
② 노출은 재설계 필요
광고 노출이 사실상 전량 KBO 종속(미니게임 포함). 올스타 브레이크 때 총 노출 −25% — 완충 장치 없음.
③ 성패 = 지면·재화
비KBO 지면 광고 구좌 신설 + 미니게임 재화 볼 전환이 대체의 필수 조건. KPI = 실제 광고매출이 계획의 70% 이상.

1대체 로드맵 — 광고 노출 재설계

실행 핵심 — 현황 진단(§4)의 "KBO 공백 = 노출 공백"을 메우는 실행 계획. 4·5·6(비KBO 지면·재화 재설계)은 시즌 종료 전 반드시.
#이니셔티브신설 노출 지면 / 작용시점
1경품 응모 시 광고 보고 볼 1회 더(1일 5회)경품 응모 지면 광고(티켓 AD 모델 이식)7월 론칭 동시
2낙첨자 케어(광고 3회·최대 30% 볼 환급)경품 낙첨 지면 광고 + 이탈 방지8월
3스페셜 응모(월단위, 광고로 추가 응모)*스페셜 경품 지면 광고 + 마케팅 후킹8월
4비KBO 예측 '광고 보고 보상 2배'비KBO 예측 지면 광고(경제·사회이슈 등)적용 가능 시 빠른 반영(8월 예상)
5광고 보고 볼 받기(1일 n회)볼 적립 전용 광고 구좌 신설적용 가능 시 빠른 반영(8월 예상)
6미니게임 재화 볼로 전환 + 볼 미니게임 신설기존 최대 노출장치(주 2.2만)를 비KBO로 전환적용 가능 시 빠른 반영(8월 예상)

* 스페셜 응모 상품 = 기보유(이전 최저가 경매용 구매) 실물 경품 활용, 응모는 정규 추첨 방식.

시퀀싱 — 1~3은 경품 트랙 강화, 4·5·6은 노출 구조 재설계로 적용 가능 시점에 즉시(8월 예상) 반영. 특히 6번(미니게임 볼 전환)은 "지금 노출의 33%를 만드는 미니게임이 응모권=티켓에 묶여 위험"을 직접 해소하므로 최우선. 4·5·6이 안 깔리면 참여가 옮겨가도 노출은 0 → 검증 자동 실패.

4현황 진단 & 검증 가설

3-1. 광고 노출 구조 — 사실상 전량 KBO 종속

픽·승부예측 52.4% (KBO 직접)
미니게임 33.1% (KBO 간접 · 응모권→티켓)
직관 티켓 응모 12.4% (KBO 직접)
팀 응원전 2.1% (야구 연계)
순수 비KBO 광고 노출 ≈ 0. 미니게임도 응모권(티켓 전용) 파밍 수단이라 KBO 간접 종속이고, 월드컵·경품 응모 지면엔 광고 구좌가 없다. 현재 광고 노출의 거의 전부가 야구 종료 시 흔들린다.

3-2. 자연 실험 A — 월드컵 (비KBO가 채울 수 있나 · 공급)

KBO 메인예측 참여월드컵 예측 참여 (주간 유저수)
75%
월드컵 예측이 KBO 규모의 (피크 6,535명)
40,717
주간 월드컵 볼 소진(경품 응모)
야구 없이도 (예측→볼→경품) 루프가 KBO 근접 규모로 작동. "이탈" 우려의 반증. 단, 이 루프의 광고 노출은 0(지면 구좌 없음) — 참여는 대체됐으나 BM은 미대체.

3-3. 자연 실험 B — 올스타 브레이크 (KBO 빠지면 · 수요)

KBO 지면 노출비KBO 표기 지면(미니게임 등) (주간 노출수)
KBO 공백 = 노출 공백. 경기 없는 구간에 KBO 지면 노출이 눌리며 총 노출이 6/22 피크 54,888 → 7/13 40,985로 −25%. 이때 "미니게임 비중 상승"은 미니게임이 커진 게 아니라 KBO 지면이 빠져 분모가 줄어든 착시 — 완충한 진짜 비KBO 지면은 없었다. 시즌 종료(픽까지 영구 소멸)는 임팩트가 더 크다.

3-4. 검증 가설

H1 · 수요 이동 월드컵 실측 지지
KBO 공백 시 유저는 비KBO 콘텐츠(월드컵 등)로 이동해 참여·DAU가 유지된다.
H2 · 노출 대체 이니셔티브 조건부
비KBO 지면·재화가 신설되면 이동한 참여가 광고 노출로 전환돼 매출을 방어한다.
H3 · 구조적 공백 확인됨
현 구조 그대로면 KBO 공백 = 노출 공백. 지면·재화 재설계 없이는 대체 불가.

5KPI — 성공 / 실패 기준

기준선 — 광고 매출 계획은 시즌 종료(9월) 후에도 성장(9월 393만 → 12월 657만)하는 전망치다. 이 성장은 야구가 빠진 뒤에도 매출이 오른다는 뜻이므로, 계획 자체가 "대체 성공 시나리오". KPI = 실제 매출이 이 계획을 얼마나 따라가는가.
✅ 성공
실제 광고 매출이 계획의 70% 이상(특히 시즌 종료 후 9~12월). 비KBO 지면이 KBO 상실분 흡수.
△ 주의
계획의 60~70%. 대체 부분적 → 이니셔티브 보강·예산 조정.
✕ 실패
계획의 60% 미만. 비KBO 노출이 KBO 상실분 미흡수 = 대체 실패.

선행 지표 (월별 체크포인트)

  • 비KBO 지면(이니셔티브 1~6) 노출 증가분이 KBO 노출 감소분을 따라오는가
  • 비KBO 예측·경품 응모 참여·볼 소진이 유지·성장하는가
  • 미니게임 볼 전환 후 미니게임 노출이 유지되는가
  • DAU/WAU·리텐션이 시즌 종료 후에도 유지되는가(H1)
임계값 — 성공선 70%는 사업부 운영 계획(광고 매출의 ~70% 운영)과 정합하는 잠정값. 최종 밴드는 데이터팀·사업부 합의로 고정.

6확인 · 후속

  • 미니게임 노출 유지 이유 — 올스타 브레이크에도 유지된 게 응모권 비축(→시즌 종료 시 무너짐)인지, 게임 재미인지. 6번 이니셔티브 시급성 근거
  • 브레이크 구간 지면별·D-14 정밀 분해 — KBO 지면별 감소폭 실측 → 시즌 종료 시 필요한 비KBO 노출 신설 규모 역산
  • 성공/실패 임계값 확정 — 데이터팀·사업부 합의
  • KBO 시즌 종료 예정일 — 검증 측정 구간(before/after) 설정
  • 선결 과제 — 이니셔티브 4·5·6을 적용 가능 시점(8월 예상)에 반영. 미이행 시 검증 자동 실패
