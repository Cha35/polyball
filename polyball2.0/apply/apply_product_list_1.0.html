0목적

경품 응모는 KBO 직관 티켓에 이은 두 번째 보상 트랙이다. 목적은 두 가지 — ① 야구 시즌에 종속된 광고 매출 리스크를 방어하고, ② 볼(P) 사용을 촉진해 광고 노출(BM)을 확대하는 것. 본 문서는 시즌 종료 대비 전략과 론칭 로드맵(1차 배포 → 낙첨 케어 → 확정당첨 → 스페셜 응모)을 정의한다.

경품 응모의 역할 (연쇄 차단)

경품은 볼(P)로 돌아가고, 볼은 야구 외 예측 콘텐츠(월드컵·이스포츠·연예·이슈 등)에서도 발생한다. 따라서 티켓이 사라져도 "볼을 모으려 광고를 보는" 루프가 유지되면 광고 노출·매출이 방어된다. 관건은 시즌 종료 전에 경품 응모 습관을 심어두는 것.

① 볼 획득 경로 다변화
야구 외 예측 콘텐츠에서 볼 발생 유지 → 시즌 무관하게 볼 공급이 끊기지 않도록 (운영PRD 카테고리 확장과 연계)
② 경품 응모 상시 운영
"광고 볼 이유"를 티켓 → 경품으로 이전. 티켓이 담당하던 리텐션·광고 동기를 경품이 승계
③ 시즌 종료 前 습관 형성
스페셜 응모로 론칭 초기에 조기 후킹 → 8~9월 이탈 국면 전에 경품 참여율·재방문을 끌어올려 티켓 이탈분을 흡수
④ 경품 루프 내 광고 지점 유지
응모완료 "광고 보고 1번 더"(1일 5회) + 낙첨 환급 광고 → 경품 활성화가 곧 광고 노출·매출 방어
관리 지표 — 시즌 종료 시점(9월)에 경품 응모 활성·재방문 유저가 티켓 이탈분을 흡수할 수준으로 올라와 있어야 한다. 8~9월 경품 주간 참여율·재방문율·볼 소진량을 선행 지표로 모니터링하고, 티켓 대비 광고 노출 이전율을 추적한다.

3스페셜 응모 (초반 후킹 · 마케팅)

론칭 초기 인지·유입·습관 형성을 위한 1개월 한정 이벤트. 고가 상품으로 화제성을 만들어 경품 응모로의 초기 유입을 극대화하고, 시즌 종료 대비 경품 습관 형성(1항 ③)의 첫 단추 역할을 한다.

4정규 회차 상품 — 예산 2단계 (현재 / 확대)

주간 회차 정규 상품을 예산 2단계로 운영한다. CASE 1(현재 실행안)은 현 광고 매출 수준에 맞춘 기본안, CASE 2(확대안)는 광고 매출 증가 시 동일 경품의 당첨 인원을 늘리는 확장안이다. 발동 기준은 주간 광고 매출이 해당 상품비를 안정적으로 상회하는 시점.

CASE 1 — 현재 실행안 (예산 30만원)

주간 광고 매출(~427,000원) 대비 72%. 현 매출로 지속 가능. 당첨 50명.

CASE 2 — 확대안 (예산 50만원, 광고 매출 증가 시)

동일 경품 구성에서 당첨 인원만 확대(50 → 112명). 상품비 490,100원으로 현 주간 매출 대비 115%이므로, 주간 광고 매출이 약 49만원 이상으로 성장한 뒤 발동한다.

확대 방향 — 인원 확대에서 품목 확대로 CASE 2는 동일 경품의 당첨 인원 증가 구조다. 다만 향후 확대는 인원만 늘리기보다, 유저가 선호하는·응모가 몰리는 경품을 트래킹(상품별 응모권 사용량·참여자 수)하여 경품 종류를 넓히는 방향으로도 확장한다. 인기 카테고리를 데이터로 식별해 회차 구성을 최적화하면, 같은 예산으로 참여·볼 소진(=광고 노출)을 더 끌어올릴 수 있다.

경품 당첨자를 티켓(주 ~120)보다 낮게 두는 근거

근거내용
비용 성격티켓은 구장·제휴 기반 고정 공급(현금 지출 최소), 경품은 건당 현금 구매(재원=광고 매출) → 예산 상한에 묶임
수요 대비 BM경품 수요 압도적(월드컵 경품 유니크 4,388·응모권 14.6만 장). 소량만 걸어도 경쟁·희소성 유지 → 볼 사용↑ → 광고 노출↑. 많이 주면 역효과
운영 지속성티켓은 시즌 한정, 경품은 상시 반복 지출 → 매주 광고 매출을 넘기지 않도록 보수적으로
참고 데이터 — 경품 수요(월드컵): 유니크 4,388명·응모권 146,207장 / 티켓 주간 응모: 181~530명 / 티켓 주간 당첨 상한: ~120명. 경품 수요가 티켓의 10배 이상이므로, 당첨 50명은 의도된 희소성(당첨률 낮음)이며 BM에 유리.

5확인 · 후속 필요

  • 스페셜 응모 "최저가 경매" 방식 상세 — 볼 사용·입찰·낙찰 규칙, 라운드 주기, 정규 응모/확정당첨과의 관계. SD-PZ에 별도 화면·정책 정의
  • 볼 발생량 점검 — 시즌 종료 후에도 야구 외 예측 콘텐츠에서 볼이 충분히 발생하는지(광고 노출 유지의 전제). 부족 시 콘텐츠 확장·볼 지급 정책 보강
  • 광고 매출 방어 목표치 — 시즌 종료 시점 경품 응모가 흡수해야 할 광고 노출/매출 규모를 수치 목표로 설정
  • 스페셜 응모 종료 후 연착륙 — 1개월 후킹 종료 시 참여 급감을 막을 정규 회차·확정당첨으로의 전환 설계
