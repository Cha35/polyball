운영 정책 — 제세공과금 · 수령 모델

제세공과금 = 운영위탁 (확정) — 제세공과금(5만원 초과 경품)의 신고 대행은 폴리볼이 아닌 운영위탁 업체가 진행(직관 티켓과 동일 구조). 폴리볼은 제세공과금 정보를 수집·입금 확인하지 않으며, 배송지 정보를 위탁업체에 전달하고 유저에게 "3영업일 내 안내" 노티만 제공한다. 기프티콘은 제세공과금 발생 수준의 고가 상품을 운영하지 않는 것을 전제로 한다.
개인정보처리 위탁 ≠ 제3자 제공 (v1.8 정정) — 실물경품 배송지 정보를 운영위탁 업체에 전달하는 행위는 개보법 §26 개인정보처리위탁이며 §17 제3자 제공이 아니다. 따라서 위탁 자체에 대한 별도 동의는 받지 않고, 위탁 사실만 고지한다. 유저로부터 실제 수집하는 것은 배송지주소·수령인·연락처이며, 이는 개인정보 수집·이용 동의 1건으로 받는다(목적: 경품 배송) → WL-03-3.
TBD(향후) 제세공과금 자사 직접 처리 전환 — 이번 버전은 제세공과금 신고를 운영위탁 업체가 대행하나, 향후 폴리볼이 직접 처리로 전환할 예정이다. 전환 시점에 수집 항목·동의·정산 정책을 이 문서에 추가한다(현재 버전 범위 아님).
수령 모델 (v1.5 갱신) — 기프티콘은 발행분 정산 대상이라 [결과 확인하기] 클릭 시점(RC-01)에 발급/코드 할당까지 동기 처리한다(발급 실패 시 결과 확인 화면 자체에 도달하지 않고 재시도). 실물경품은 [배송지 입력]으로 수령 의사를 받는다(변경 없음). 수령 기한은 당첨 발표일로부터 7일이며, 기한 내 미수령 시 당첨 자동 취소(복구 불가). 어드민 PZ는 '발급 예약'만 보유. TBD 기프티콘은 결과 확인=수령이 사실상 같은 행위가 되므로, "7일 미수령"의 정의(=7일 내 결과 확인하기 미클릭)를 문서 전반에 재확인 필요.
