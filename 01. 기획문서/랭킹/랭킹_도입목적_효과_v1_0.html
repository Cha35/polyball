핵심 전제
두 경로 모두 "광고 시청 기회의 수"가 늘어야 수익이 늘어납니다. 기회를 늘리는 유일한 방법은 픽 참여 횟수와 응모 세션 수를 늘리는 것입니다. 랭킹은 이 두 가지를 늘리기 위한 설계입니다.
02 도입 목적
1
픽 참여 횟수 증가 → 2배 RV 노출 기회 확대
현재 유저는 관심 있는 경기만 선택적으로 참여합니다. 랭킹 기준이 적중 수 절대값이기 때문에, 순위를 올리려면 더 많은 마켓에 참여해야 유리합니다. 참여 횟수가 늘수록 적중 기회가 늘고, "보상 2배 버튼" 노출 횟수도 늘어납니다. 버튼 노출 횟수 × 유저 CTR = 실제 광고 시청 수로 직결됩니다.
2
재방문 루프 생성 → 응모 세션 수 증가
현재 재방문 트리거는 당첨 결과 확인(월요일)과 외부 캠페인에 의존합니다. 일간 분석 데이터에서도 알림톡·인플루언서 투입 없는 날 DAU가 급감하는 패턴이 반복됩니다. 랭킹이 생기면 "내 순위가 어떻게 됐지"라는 이유로 앱을 다시 열게 되고, 이 재방문이 응모 탭 진입으로 이어지면 응모 완료 RV 노출 기회가 추가로 생깁니다. 외부 비용 없이 작동하는 내부 재방문 동인입니다.
3
보상(응모권)의 광고 순환 구조 설계
2·3위 보상인 응모권 50장은 유저를 다시 응모 화면으로 유인합니다. 응모 완료 후에는 추가 RV 시청 기회가 노출됩니다. 랭킹 보상 자체가 광고 노출 기회를 만드는 순환 구조입니다. 1위 확정 티켓은 서비스 내 희소성 자원으로 신규 유입과 구전 효과를 기대합니다.
03 예상 효과
직접 효과
2배 RV 버튼 노출 기회 증가
픽 참여 횟수 증가 → 적중 건수 증가 → 버튼 노출 횟수 증가. 실제 광고 매출은 버튼 노출 횟수 × 유저 CTR로 결정됩니다.
간접 효과
주중 재방문율 개선
외부 자극 없이 작동하는 자체 재방문 동인 확보. 재방문 → 응모 → 응모 완료 RV 노출 기회 추가 생성.
간접 효과
보상 → 응모 순환 구조
2·3위 응모권 50장 → 추가 응모 세션 → 응모 완료 RV 기회. 보상이 광고 노출을 만드는 선순환.
직접 효과
확정 티켓 희소성 → 신규 유입
"실력으로 얻는 티켓"이라는 포지셔닝이 구전 효과로 이어질 가능성. 신규 유입 → 광고 노출 풀 확대.
광고 매출 증가 공식
광고 매출 증가 = (추가 버튼 노출 횟수) × (유저 CTR)
추가 버튼 노출 = (참여 증가분) × (적중률)
광고 노출은 자동 증가가 아닙니다. 참여 횟수가 늘어야 버튼 노출 기회가 늘고, 유저가 실제로 버튼을 눌러야 매출이 발생합니다. CTR 추적이 효과 검증의 핵심 지표입니다.

05 한계 및 주의사항
⚠ 과대 해석 방지
  • 광고 노출은 자동 증가가 아닙니다. 랭킹이 만드는 것은 "광고 시청 기회"이며, 실제 매출은 유저가 버튼을 누르는 비율(CTR)에 달려 있습니다.
  • 유저 풀이 작으면 랭킹 경쟁 자체가 형성되지 않습니다. 현재 주간 활성 유저 81명 수준에서는 랭킹보다 확정 티켓 보상을 전면에 내세워 참여를 유도하는 것이 중요합니다. 경쟁 긴장감은 유저 수 300~500명 이상일 때 본격 형성될 것으로 예상합니다.
  • CTR 변화를 반드시 추적해야 합니다. 참여 횟수가 늘어도 유저가 버튼을 누르지 않으면 매출로 이어지지 않습니다. CTR이 하락한다면 보상 설계를 재검토해야 합니다.
  • 보상 구조는 주단위로 모니터링하며 재검토합니다. 초기 확정 티켓 보상이 지속 가능한지, 4위 이하 공백으로 이탈이 발생하지 않는지 지속 관찰이 필요합니다.
