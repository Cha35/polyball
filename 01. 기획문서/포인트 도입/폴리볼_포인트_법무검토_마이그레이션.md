# 폴리볼 포인트 제도 — 법무 검토 마이그레이션
> 작성 기준: Phase 2.0 포인트 설계 착수 전 법무팀 가이드 정리  
> 다음 채팅에서 이 내용을 전달하면 논의를 이어갈 수 있습니다.

---

## 1. 법무팀이 제시한 두 가지 원칙

1. **포인트는 무료 획득만 가능**
   - 현금 구매·충전·환전 불가
   - 유저가 금전을 지불하는 방식의 포인트 획득 전면 금지

2. **포인트 → 재화 교환은 응모 형태로만 가능**
   - 포인트 N개 = 상품 확정 수령 구조 불가
   - 포인트 N개 = 응모권 N장 (추첨 참여) 구조는 허용

---

## 2. "대가성"의 정의 — 최종 정리

**대가성 판단 기준:**
> 포인트를 얻기 위해 유저가 수행한 행동에, **제3자(광고주)의 실제 금전이 직접 연동**되어 있는가

- 대가성 **없음**: 픽 참여, 출석, 서비스 내 행동 (플랫폼 자체 비용)
- 대가성 **있음**: 오퍼월 (광고주가 행동당 CPI/CPA 단가 계약)

**핵심 법적 프레임:**
사행성 판단은 **대가성 단독**이 아닌 **대가성 × 우연성의 결합**으로 판단됩니다.

| 구조 | 대가성 | 우연성 | 위법 여부 |
|---|---|---|---|
| 대가성 포인트 → 확정 교환 | 있음 | 없음 | 합법 (앱테크) |
| 무료 포인트 → 확정 교환 | 없음 | 없음 | 합법 |
| 무료 포인트 → 응모 | 없음 | 있음 | 합법 ✅ (폴리볼 현재) |
| 대가성 포인트 → 응모 | 있음 | 있음 | **위법 소지** ❌ |

폴리볼이 응모(우연성) 구조를 유지하는 한, 포인트에 대가성이 붙으면 경품류 고시 위반 또는 사행행위 소지가 생깁니다.

---

## 3. 오퍼월이 제외되는 이유

오퍼월(앱 다운로드, 특정 과제 달성 등 CPI/CPA 구조)은 아래 이유로 제외됩니다.

- 광고주가 유저의 특정 행동에 단가를 직접 설정
- 포인트 1개당 원화 가치가 구조적으로 역산 가능 → 포인트가 재산적 가치를 지님
- 이 포인트로 응모 참여 시 = "돈을 내고 경품 추첨에 참여"하는 것과 실질 동일

일반 앱테크(캐시슬라이드 등)가 합법인 이유는 **확정 교환** 구조(우연성 없음)이기 때문이며, 응모 구조인 폴리볼과는 다릅니다.

---

## 4. RV(리워드 비디오) — 미확인 잔여 사항

RV도 "광고주 돈 → 포인트" 흐름이라는 점에서 대가성 논란 여지가 있습니다.  
다만 **현재 서비스에서 RV → 응모권 구조가 이미 운영 중**이며, 법무팀이 현행 구조를 승인한 것을 사실상 허용으로 간주하고 진행합니다.

> ⚠️ 기회가 되면 법무팀에 명시적 확인 권장:  
> "RV 보상 포인트는 대가성으로 보는가, 플랫폼 자체 재량으로 보는가?"

---

## 5. 포인트 획득 루트 — 확정 범위

| 구분 | 허용 여부 | 비고 |
|---|---|---|
| 픽(예측) 참여 | ✅ 허용 | 플랫폼 자체 보상 |
| RV 광고 시청 | ✅ 허용 (현행 운영 중) | 법무 명시 확인 권장 |
| 출석 / 서비스 내 행동 | ✅ 허용 | |
| 이벤트·미션 보상 | ✅ 허용 | |
| 오퍼월 (CPI/CPA) | ❌ 제외 | 대가성 + 응모 결합 위법 소지 |
| 현금 구매·충전 | ❌ 제외 | 법무팀 원칙 1 위반 |
