01 기능 발의 배경 및 본질 정의
직관 기록 기능이 왜 시작됐고, 무엇을 해야 하는지에 대한 팀 내 공유 정의.
발의 배경

폴리볼 앱 내 승부 예측·응모 이외 콘텐츠가 없다는 문제의식에서 출발했다. 타사 서비스 벤치마킹 과정에서 야구보구·직관일기·직관로그 등 전문 기록 앱이 다수 존재함을 확인했으며, 동시에 인스타그램 스토리를 통한 직관 자랑 행동과 커뮤니티에서 '승요/패요'라는 승률 기반 소통 패턴을 발견했다.

이에 단순 기록 앱이 아닌, 폴리볼이 공신력을 부여하는 인증 기반 아카이빙 방향으로 논의를 시작했다.

기능 본질 정의 (최종)

"인증된 직관 경험이 공신력 있는 카드로 발행되어 내 지갑에 쌓이는 것"

단순한 기록 앱이 아니라, 폴리볼이 공식 인증한 팬 경험을 카드로 발행하여 내 지갑에 자산으로 축적시키는 구조다. 나아가 내 지갑이 독립 탭으로 확장되고, 장기적으로 월렛 서비스로 진화하는 방향의 첫 번째 레이어다.

키워드의미없으면 어떻게 되나
인증셀프 등록이 아닌 폴리볼이 확인해준 공신력공신력 없음 → 자랑할 근거 없음 → 소셜 확산 불가

내 지갑카드가 쌓이는 공간. 당첨자 전용에서 모든 유저로 확장비당첨 유저에게 의미 없는 빈 공간 그대로 유지
PO 핵심 피드백
"차별화가 되려면 단순히 내가 셀프로 등록하는 게 아니라 공신력 있게끔 해줘야 함. 직관 기록을 기반으로 소통(자랑/선망)하게끔 해주면 차별화 요소가 될 수 있음. BM: (단기) 사진 등록 시 광고 노출 → (장기) 시즌 종료 시 직관 기록북/굿즈 판매 등 검토 가능."
— 남영광 PO

핵심 논의 결론
세션 결론 인증은 강제가 아닌 선택 + 등급 차등 구조가 맞다
인증 허들이 높으면 참여율이 떨어지고, 인증이 없으면 공신력이 없다는 딜레마에서 출발했다. 결론은 "인증 없이도 기록 가능, 인증하면 카드 등급 상승"이라는 선택적 다단계 구조였다.
인증 강도에 따라 카드 등급 차등 부여. 등급 높을수록 NFT 전환 시 더 높은 가치. 단, 인증 미완료 카드와 완료 카드는 시각적으로 명확히 구분.
04 1차 FGI 인사이트 (개별 인터뷰 5인)
상상 프로토타입을 사용한 5인 개별 심층 인터뷰에서 도출된 인사이트.
인사이트 #1 인증의 가치 전달이 먼저다 높음
인증을 귀찮아하는 유저도 "왜 인증해야 하는지"를 이해하면 태도가 바뀌었다. 인증 화면 진입 전, "폴리볼이 인증하면 뭐가 달라지는지"를 한 줄로 명확하게 보여주는 것이 선행되어야 한다.
"인증 화면이 그냥 나오니까 왜 해야 하는지 몰랐어요. 근데 '폴리볼이 공식 확인한 기록이 된다'고 하니까 해야겠다는 생각이 들었어요."
— 김민준 (33세)
인증 진입 전 한 줄 안내 UI 추가: "인증하면 폴리볼 공식 마크가 붙어요. 자랑할 때 달라요." 설명 읽어야 아는 구조는 누구도 읽지 않음.
인사이트 #2 NFT라는 단어는 걸림돌이다 높음
5명 중 3명이 NFT 단어에 부정적이거나 의미를 몰랐다. 2021~2022년 붐 이후 사기 이미지가 남아있어 NFT를 전면에 내세우는 것은 진입 장벽이 된다.
"NFT라는 단어는 좀 거슬렸어요. 사기 느낌이라고 해야 하나. 근데 '영구 보존하기'라고 하면 해볼 것 같아요."
— 이수진 (29세)
전면 표현 변경: "NFT로 만들기" → "공식 카드로 만들기" 또는 "영구 보존하기". NFT는 기술 설명 레이어에만 사용.
인사이트 #3 인증은 강제가 아닌 선택이어야 한다 높음
인증 없이도 기록 가능하되 인증하면 더 좋은 카드가 나오는 구조가 이탈을 막는다. 강제 인증은 개인 아카이빙 목적 유저를 완전히 잃는다.
"인증이 선택이면 쓸 것 같아요. 강제면 안 쓸 것 같고요. 나는 남한테 보여주려는 게 아니니까, 내 기록은 내가 알면 되는데 인증을 강요하면 이상하잖아요."
— 최재원 (31세)
인증 미완료 상태로도 기록 저장 및 카드 발행 가능. 단, 인증 카드와 비인증 카드의 시각적 구분 필수. 소셜 공유 시 인증 마크 유무가 자연스럽게 보이도록 설계.
인사이트 #4 기록 타이밍은 경기 후가 더 자연스럽다 중간
경기 중 인증보다 경기가 끝나고 구장을 나오면서 또는 집으로 오는 지하철에서 기록하는 패턴이 반복적으로 나타났다. 경기 중에는 핸드폰을 보기 어렵고, 경기 직후는 흥분 상태라 즉시 하기도 어렵다.
푸시 알림 발송 타이밍: 경기 종료 직후 또는 30분~1시간 후. 유저가 설정 가능하도록 "경기 종료 직후 / 1시간 후 / 다음날 아침" 옵션 제공. 기본값은 경기 종료 1시간 후.
인사이트 #5 소셜 핵심은 랭킹과 연결이다 중간
승요/패요 랭킹, 같은 경기 간 사람들 연결, 같이 간 친구와의 공동 기록 세 가지가 반복적으로 언급됐다. 커뮤니티 자체보다 데이터 기반 연결이 핵심이다.
MVP에서는 프로필 뱃지(직관 횟수/승률)로 시작. 커뮤니티 구축 후 랭킹 → 공동 기록 → 같은 경기 연결 순으로 단계별 확장.
인사이트 #6 카드 공유용 템플릿이 바이럴의 핵심이다 중간
인스타 스토리에 올리고 싶다는 반응이 강했다. 내 사진이 카드 배경으로 들어가고, 승률과 폴리볼 인증 마크가 함께 보이는 구조가 가장 선호됐다.
인스타 스토리 사이즈에 최적화된 카드 공유 템플릿 제공 필수. 카카오톡 이미지 공유, URL 복사도 함께 지원. 단일 채널(인스타)에만 맞추면 카카오톡 중심 유저 이탈.

카드 공유 템플릿이 바이럴 핵심✅ 확인됨다채널 공유 + 시즌 통계 카드 아이디어 추가
2차 신규 인사이트
인사이트 #7 폴리볼만의 차별점은 예측과 직관의 연결이다 높음
다른 기록 앱이 절대 할 수 없는 것이 바로 이것이다. 같은 경기를 예측하고 직관 간 사람들을 연결하는 기능은 폴리볼 데이터가 있어야만 가능하다. 소셜 기능 설계의 중심이 되어야 한다.
"오늘 LG 이긴다고 예측한 사람들 중 직관 간 사람들끼리 연결되면 어떨까요? 이건 폴리볼만 할 수 있는 거잖아요."
— 윤재훈 (32세)
직관 기록 카드에 예측 결과 자동 연동. "예측하고 직관 간 사람들" 필터 기반 커뮤니티 기능 장기 로드맵에 포함.
인사이트 #8 기록 동기가 사람마다 완전히 다르다 높음
아카이빙(남도현), 자랑(김서연), 친구와 연결(박혜진), 자산화(이정호), 서비스 내 혜택(윤재훈). 5명의 동기가 모두 달랐다. 하나의 동기만 겨냥하면 나머지 유저를 잃는다.
기록 동기별 진입점 다르게 설계 검토. 예: "나만의 기록" 탭 / "자랑하기" 공유 버튼 강조 / "친구 연결" 동행인 태그 / "자산 쌓기" 카드 등급 강조. MVP에서는 단일 플로우로 시작하되, 온보딩에서 동기 파악 후 화면 강조점 다르게 노출 고려.
인사이트 #9 시즌 통계 카드가 바이럴 최고 무기다 중간
연말 시즌 통계 카드에 대한 반응이 그룹 전체에서 가장 뜨거웠다. 승요/패요 판정 + 팀 컬러 배경 + 주요 통계가 합쳐지면 자발적 공유 가능성이 매우 높다. Spotify Wrapped와 유사한 자발적 바이럴 효과 기대.
"'당신은 2026 시즌 승률 80%의 승리요정입니다' 이런 거요. 팀 컬러로 배경도 바뀌고, 저 이거 바로 올릴 것 같은데요."
— 김서연 (24세)
시즌 Wrapped 기능 로드맵 포함. 필수 정보: 총 직관/집관 횟수, 직관 승률, 승요/패요 판정, 가장 많이 간 구장, 응원팀 컬러 배경. 인스타·카카오·링크 복사 공유 지원.
인사이트 #10 공유 채널을 인스타에만 맞추면 안 된다 중간
인스타 스토리, 카카오톡 공유, 링크 복사. 유저마다 주요 공유 채널이 달랐다. 인스타 최적화에만 집중하면 카카오톡 중심 유저를 잃는다.
공유 기능 구현 시 인스타 스토리 사이즈 + 카카오톡 이미지 + URL 링크 복사 세 가지 모두 지원. 향후 SNS 채널 추가에 유연하게 대응할 수 있는 구조로 개발.

폴리볼 직관 기록 설계 적용 인사이트
데이터 #1 직관 권유/피권유 → 카드 공유가 권유 도구가 될 수 있다 높음
전체 권유 경험 평균 3.27점, 피권유 3.11점(4점 만점)으로 야구 직관은 이미 입소문 기반으로 퍼지는 경험이다. 삼성(3.33점)·두산·SSG(3.29점)이 권유 경험이 특히 높았다.
권유/피권유 행동이 이미 오프라인에서 일어나고 있다면, 폴리볼 인증 카드가 그 권유의 도구가 될 수 있다. 삼성·두산·SSG 팬 커뮤니티를 초기 바이럴 타겟으로 우선 공략 검토.
데이터 #2 91.5%가 동행인 있음 → 동행인 태그가 핵심 기능이어야 한다 높음
혼자 직관하는 비율은 8.5%에 불과하다. 직관의 91.5%는 누군가와 함께 가는 경험이며, 가족 동행 비율은 2024년 36.1%에서 2025년 43.7%로 증가 추세다.
동행인 태그 기능을 부가 기능이 아닌 핵심 기능으로 격상. 같이 간 사람이 폴리볼 유저라면 카드 연결, 비유저라면 초대 링크로 연결되어 유저 획득 채널로도 활용 가능.
데이터 #3 평균 15.4회 직관 → 승률 오픈 기준 10개로 상향 검토 중간
시즌 중 평균 15.4회 직관이라면, 기록 5개 기준은 시즌 초반에 너무 빨리 달성될 수 있다. 기준이 너무 낮으면 통계가 충분히 쌓이기 전에 열리는 문제가 발생한다.
직관 승률 오픈 기준을 5개에서 10개로 상향 검토. 10개면 약 5~6주 소요로 적절한 긴장감이 유지됨. 단, 집관 기록까지 합산 여부는 별도 결정 필요.
데이터 #4 티켓 구매 채널 다양성 → 인증 방식 다각화 필요 중간
공식 사이트 구매는 전체 69.7%이지만, 한화(40.3%)는 현저히 낮고 중고거래 비율(27.6%)이 높다. OCR 기반 티켓 인증은 공식 예매 유저만 커버한다.
OCR 티켓 인증은 약 70%를 커버하는 기본 방식으로 설계. 한화 팬처럼 비공식 경로로 입장하는 유저를 위해 "경기 시간 한정 현장 인증 버튼" 또는 GPS 인증(고도화)으로 보완.
데이터 #5 10년+ 장기 팬 37% → 과거 기록 소급 입력 v2 일정 앞당기기 중간
10년 이상 장기 팬이 37.2%로 가장 많은 비율을 차지한다. 이 유저군은 "내가 10년간 갔던 경기도 기록하고 싶다"는 소급 입력 니즈가 강할 것으로 예상된다. LG(53.8%)는 절반 이상이 10년 이상 장기 팬이다.
MVP에서 제외된 과거 기록 소급 입력 기능의 v2 일정을 너무 늦추지 않도록 조정 검토. 특히 LG·롯데·KIA 팬 베이스에서 니즈가 높을 것으로 예상.
데이터 #6 여성 관람객 56.7% → 카드 디자인 기준을 여성 유저로 중간
전체 여성 관람객 비율이 56.7%로 남성을 초과한다. 두산(62.2%), SSG(61.8%), KIA(59.0%) 순으로 여성 비율이 특히 높다. FGI 시뮬레이션에서도 다꾸·포토카드 감성으로 접근했을 때 반응이 좋았다.
기록 카드 디자인 방향성 설정 시 여성 유저를 기준으로 삼는 것이 타당. 포토카드 수집 감성, 배경 커스터마이징, 팀 컬러 적용 등 감성 요소 우선 강화.
