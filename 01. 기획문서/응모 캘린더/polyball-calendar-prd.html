01 개요 및 배경

폴리볼은 승부 예측 콘텐츠를 통해 응모권을 획득하고, 획득한 응모권으로 직관 경기에 응모해 티켓을 받을 수 있는 서비스다.

현재 응모는 주간 단위로 운영되며, 기존 응모 페이지에서 해당 주의 응모 가능 경기를 확인할 수 있다. 그러나 유저가 과거 응모 이력, 현재 진행 중인 응모, 미래 예정 경기를 한눈에 파악할 수 있는 전용 뷰가 없어, 응모 참여 동기 및 서비스 재방문율을 높이는 데 한계가 있다.

이에 따라 응모 캘린더 전용 페이지를 신규 개발하여 유저의 응모 여정 전반을 시각적으로 제공하고자 한다.

02 목표
Goals
  • ✅유저가 응모 가능한 경기를 캘린더 형태로 직관적으로 파악할 수 있게 한다
  • ✅과거 응모 참여 이력 및 당첨 결과를 캘린더에서 바로 확인할 수 있게 한다
  • ✅미래 예정 경기에 대한 알림 설정으로 응모 참여율을 높인다
  • ✅응원 팀 필터를 통해 개인화된 응모 경험을 제공한다
  • ✅응모권 개념을 캘린더 내에서 자연스럽게 학습할 수 있는 경험을 제공한다
Non-Goals
  • ❌기존 응모 페이지(주간 응모 진행)를 대체하지 않는다
  • ❌승부 예측 콘텐츠 자체의 UI/UX는 이번 범위에 포함하지 않는다
  • ❌KBO 외 타 리그/종목은 이번 범위에 포함하지 않는다 (향후 확장 가능성 열어둠)
03 응모 구조 이해

캘린더를 설계하기 위해 기반이 되는 응모 운영 구조는 아래와 같다.

항목내용
응모 기간매주 월요일 ~ 일요일
응모 대상 경기응모 기간 기준 +2주 후 화요일 ~ 일요일 경기
응모 가능 구단LG, KT, SSG, NC, KIA, 롯데, 키움 홈경기 (7개 구단)
응모 불가 구단두산, 한화, 삼성 홈경기 — 티켓 미보유, 현재 준비 중
시즌 범위2026 KBO 시즌
04 사용자 시나리오
Scenario A
응모 가능한 경기를 찾아 응모한다 유저가 캘린더를 열어 현재 응모 가능한 경기를 확인하고, 원하는 경기를 탭해 응모 페이지로 진입한다.
Scenario B
내가 응모한 경기의 결과를 확인한다 유저가 캘린더에서 과거 날짜를 탐색하며 자신이 응모했던 경기를 찾고, 탭하여 당첨 여부를 확인한다.
Scenario C
응원 팀의 다음 응모 경기를 미리 파악한다 유저가 응원 팀 필터를 적용해 특정 구단의 경기만 필터링하고, 미래 예정 경기에 알림받기를 설정한다.
Scenario D
월간 일정을 한눈에 파악한다 유저가 월간 뷰로 전환하여 이번 달 전체 경기 일정과 응모 상태를 파악한다.
Scenario E
응모 불가 구단 팬이 캘린더를 탐색한다 응모 불가 구단(두산/한화/삼성) 팬이 응원팀 필터 적용 시, 빈 화면 대신 '준비 중 안내 + 알림받기' 화면을 만나고 이탈 없이 알림을 설정한다.
05 기능 요구사항
5-1 응모권 현황 바 (상단 고정)

유저가 캘린더를 탐색하는 내내 응모권 보유 현황을 인지할 수 있도록 상단에 고정 노출한다.

유저 상태노출 내용
응모권 보유잔여 응모권 수량 표시
응모권 미보유'승부 예측하고 응모권 받기' CTA 버튼으로 전환 → 예측 페이지 이동
💡 응모권 수량은 캘린더 진입 시 + 응모 완료 직후 갱신. 미보유 상태 CTA는 승부 예측 페이지로 연결되어 응모 학습 진입점 역할을 함.
5-2 캘린더 뷰
기능설명
주간 뷰기본값. 해당 주의 경기를 날짜별로 표시
월간 뷰월 단위 그리드. 경기 수를 닷(dot)/숫자로 요약. 날짜 탭 시 당일 경기 리스트 시트 노출
뷰 전환주간 ↔ 월간 탭으로 전환 가능
날짜 이동이전/다음 주(월) 스와이프 또는 화살표
날짜 피커헤더 연/월 탭 시 날짜 피커로 빠른 이동
오늘로 이동현재 날짜로 즉시 이동 버튼
스크롤 위치 복원경기 상세 진입 후 복귀 시 이전 스크롤 위치 복원 (세션 저장)
5-3 경기 상태 표시

경기 상태는 텍스트 레이블 뱃지로 명시적으로 표현한다. 최초 진입 시 범례를 노출하고, 확인 후 자동 숨김 처리한다.

상태정의표현 방식
응모중현재 응모 기간, 응모 열림컬러 강조 + '응모중' 뱃지
마감응모 기간 종료, 발표 전흐리게 + '마감' 뱃지
결과당첨 결과 확인 가능흐리게 + '결과' 뱃지
예정응모 미오픈 미래 경기흐리게 + '예정' 뱃지
준비중응모 불가 구단 경기회색 처리 + '준비중' 뱃지

5-5 필터
필터설명
응원 팀 필터유저가 설정한 응원 팀의 경기만 표시

⚠️ 응모 불가 구단 팬이 필터 적용 시 빈 화면 노출 금지. '현재 OO 구단의 응모는 준비 중이에요' 안내 메시지 + '응모 오픈 알림받기' 버튼 + '전체 경기 보기' 버튼 제공.
5-6 알림
알림 유형대상발송 시점
응모 오픈 알림알림받기 설정 유저응모 오픈 당일 오전
응모 마감 임박 알림해당 경기 응모 유저마감 하루 전
당첨 결과 알림해당 경기 응모 유저결과 확정 즉시
💡 알림받기 버튼 탭 시 앱 푸시 허용 여부 확인. 미허용 상태면 푸시 허용 유도 시스템 팝업 자연스럽게 노출. 발송 시점 상세 명세는 별도 알림 기획 문서로 분리.
