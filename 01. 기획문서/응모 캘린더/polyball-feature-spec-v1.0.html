01문서 개요

본 문서는 폴리볼 응모 캘린더 기능의 상세 기능 정의서로, PRD v1.2를 기준으로 각 화면 및 기능 단위의 동작 방식, 조건, 예외 처리를 정의한다. 우선순위는 P0(필수) · P1(권장) · P2(선택)으로 구분한다.
